在html5中 对于

```html
<input type="file">
```

可以用accept属性指定接收的文件类型。

### Snytax

```html
<input accept="file_extension|audio/*|video/*|image/*|media_type">
```

To specify more than one value, separate the values with a comma (e.g.

```html
<input accept="audio/*,video/*,image/*" />
```
